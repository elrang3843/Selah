1. Python 설치 필수 — MR 추출

스템 분리(MR 추출) 기능에 Python 3.10 이상이 필요합니다.
Python 3.10 or later is required for stem separation (MR extraction).

⚠ Windows 스토어(Microsoft Store) Python은 사용하지 마세요.
스토어 버전은 백그라운드 실행 시 제대로 동작하지 않습니다. 반드시 python.org 에서 설치하세요.

설치 방법

  1. python.org/downloads 에서 Python 3.10 이상(최신 안정 버전 권장)을 다운로드합니다.
  2. 설치 프로그램 첫 화면에서 "Add Python to PATH" (또는 "Add python.exe to PATH") 옵션을 반드시 체크합니다.
  3. 설치를 완료한 뒤 셀라를 재시작합니다.

설치 확인 — 명령 프롬프트(CMD) 또는 PowerShell에서:

python --version

Python 3.x.x가 출력되면 정상입니다. 오류가 나면 PATH 설정을 확인하세요.

2. ONNX Runtime 설치 필수 — MR 추출

AI 모델 추론(inference)에 ONNX Runtime 및 관련 패키지가 필요합니다.
ONNX Runtime and related packages are required to run the AI separation model.

설치 방법

명령 프롬프트(CMD) 또는 PowerShell을 열고 다음 명령을 실행합니다:

pip install onnxruntime numpy scipy

pip 명령을 찾지 못하는 경우 아래 명령을 사용하세요:

python -m pip install onnxruntime numpy scipy
⚠ 가상 환경(venv, conda 등)을 사용 중이라면 셀라가 해당 환경을 자동으로 인식하지 못할 수 있습니다. 가상 환경 없이 전역(global) Python 환경에 설치하거나, 셀라 실행 전에 가상 환경을 활성화하세요.

설치 확인:

python -c "import onnxruntime; print(onnxruntime.__version__)"

3. noisereduce 설치 필수 — 노이즈 감소

편집 > 노이즈 감소 기능에 noisereduce 라이브러리가 필요합니다. 모델 파일 없이 pip 한 줄로 설치됩니다.
The "Reduce Noise" feature requires the noisereduce library. No model file — install with one pip command.

설치 방법

명령 프롬프트(CMD) 또는 PowerShell을 열고 실행합니다:

pip install noisereduce

pip을 찾지 못하면:

python -m pip install noisereduce

설치 확인:

python -c "import noisereduce; print('noisereduce OK')"
⚠ 가상 환경(venv) 사용 시 주의
pip 출력에 .\\.venv\\ 또는 site-packages 경로가 보인다면 가상 환경에 설치된 것입니다. 셀라는 가상 환경을 자동 감지하지 않으므로 전역(global) Python에 재설치해야 합니다.

전역 Python에 설치하는 방법: 
py -m pip install noisereduce
(py 명령은 Windows Python Launcher로 전역 Python을 직접 호출합니다.)

4. audio-separator 설치 권장 — 보컬 분리

스템 분리 (보컬/반주) 기능에 audio-separator를 사용하면 Demucs보다 훨씬 넓은 범위의 목소리를 정확하게 분리할 수 있습니다.
남성·여성·어린이·비서양권 창법 등 모든 음역대의 목소리를 인식합니다.
Using audio-separator for stem separation gives far better vocal coverage than Demucs — it recognizes voices of all ranges, genders, and vocal styles.

설치 방법

명령 프롬프트(CMD) 또는 PowerShell을 열고 실행합니다:

pip install "audio-separator[cpu]"

pip을 찾지 못하면:

python -m pip install "audio-separator[cpu]"

GPU 가속을 원한다면 [cpu] 대신 [gpu]를 사용하세요. (CUDA 12.x + PyTorch 필요)

설치 확인:

python -c "import audio_separator; print('audio-separator OK')"

모델 다운로드

모델 파일(UVR-MDX-NET-Voc_FT.onnx, ~65 MB)은 셀라 모델 관리자에서 설치하거나, 처음 분리 실행 시 자동으로 다운로드됩니다.

모델은 ~/.cache/audio-separator/ 폴더에 캐시됩니다.

⚠ 가상 환경(venv) 사용 시 주의
가상 환경에 설치된 경우 셀라가 인식하지 못합니다. 전역(global) Python에 설치하세요: 
py -m pip install "audio-separator[cpu]"

5. FFmpeg 설치 필수 — MP3·MP4 임포트 & MR 추출

MP3, MP4, MKV 등 비(非)WAV 파일 임포트와 MR 추출(오디오 디코딩/인코딩)에 FFmpeg가 필요합니다.
FFmpeg is required for importing non-WAV files and for audio decode/encode during MR extraction.

방법 A — winget 사용 (가장 간단, Windows 10/11)

명령 프롬프트를 관리자 권한으로 열고 다음 명령을 실행합니다:

winget install Gyan.FFmpeg

설치 후 명령 프롬프트를 새로 열고 셀라를 재시작하세요.

방법 B — 수동 설치

  1. ffmpeg.org/download.html → Windows builds 링크에서 최신 빌드를 다운로드합니다.
  2. 압축을 해제하고, 내부의 bin\ 폴더 경로(예: C:\ffmpeg\bin)를 시스템 환경 변수 PATH에 추가합니다.
    • 시작 → "환경 변수" 검색 → "시스템 환경 변수 편집" → Path → 편집 → 새로 만들기
  3. 명령 프롬프트를 새로 열고 확인합니다:
ffmpeg -version

6. 분리 모델 다운로드 필수 — MR 추출

위의 1~3 단계 완료 후, 셀라 내 모델 관리자에서 AI 모델 파일을 다운로드합니다. (~303 MB)
After completing steps 1–3, download the AI model from Selah's built-in Model Manager.

다운로드 방법

  1. 셀라 메뉴 → 도구 > 모델 관리자를 엽니다.
  2. 목록에서 Hybrid Transformer Demucs를 선택합니다.
  3. "모델 설치 / 다운로드" 버튼을 클릭합니다.
  4. 설치 로그에 "모델 설정 완료" 메시지가 나타나면 "새로고침" 버튼을 클릭합니다.
📥 모델 파일 크기: 약 303 MB. 다운로드 중 인터넷 연결이 유지되어야 합니다.
모델 출처: HuggingFace — MrCitron/demucs-v4-onnx (MIT 라이선스, Meta AI Research)

7. 악보 인식(OMR) 패키지 설치 필수 — 악보 이미지 가져오기

파일 → 악보 이미지 가져오기 기능에 아래 Python 패키지가 필요합니다.
Required for File → Import Sheet Music: image preprocessing, OMR recognition, and MIDI export.

한 줄 설치

pip install oemer music21 mido Pillow scipy

pip을 찾지 못하면:

python -m pip install oemer music21 mido Pillow scipy
패키지 역할
oemer광학 음악 인식(OMR) 엔진
music21MusicXML → MIDI 분석·변환
midoMIDI 파일 조작 및 GM 패치 교체
Pillow악보 이미지 전처리 (그레이스케일·이진화)
scipy이미지 노이즈 제거 필터

설치 확인:

python -c "import oemer, music21, mido, PIL, scipy; print('OMR OK')"

oemer는 첫 실행 시 자체 모델 가중치를 자동 다운로드합니다 (인터넷 연결 필요).

8. FluidSynth 설치 필수 — MIDI 합성

인식된 악보의 MIDI를 악기 오디오(WAV)로 변환하는 합성 엔진입니다. 두 단계가 모두 필요합니다.
FluidSynth converts the recognized MIDI to instrument audio. Both steps below are required.

Step 1 — 네이티브 라이브러리 설치 (libfluidsynth-3.dll)

방법 A — Chocolatey (관리자 PowerShell):

choco install fluidsynth

Chocolatey는 버전에 따라 C:\Program Files\FluidSynth\bin\ 또는 C:\tools\fluidsynth\bin\에 설치할 수 있습니다. 셀라는 두 경로를 모두 자동으로 탐지합니다.

방법 B — 공식 설치 프로그램:

  1. fluidsynth.org → Download → Windows 설치 프로그램 실행
  2. 기본 경로(C:\Program Files\FluidSynth\)로 설치합니다.
  3. 셀라가 DLL을 자동으로 탐지합니다 — PATH 설정 불필요.

Step 2 — Python 래퍼 설치

pip install fluidsynth

pip을 찾지 못하면:

python -m pip install fluidsynth
⚠ 두 단계가 모두 필요한 이유
pip install fluidsynth은 Python에서 FluidSynth를 호출하는 래퍼(ctypes 바인딩)만 설치합니다. 실제로 소리를 합성하는 libfluidsynth-3.dll은 네이티브 설치 프로그램이 제공합니다. Step 1 없이 Step 2만 설치하면 합성 시 오류가 발생합니다.

설치 확인:

python -c "import fluidsynth; print('fluidsynth OK')"

문제 해결 — "FLUIDSYNTH_DLL_MISSING" 오류

합성 시 FLUIDSYNTH_DLL_MISSING 오류가 발생하면 셀라가 libfluidsynth-3.dll을 찾지 못한 것입니다. Python 패키지는 설치되어 있지만 네이티브 DLL 경로를 인식하지 못한 경우입니다.

관리자 PowerShell에서 아래 명령으로 DLL 위치를 확인하세요:

Get-ChildItem -Path C:\ -Filter "libfluidsynth*.dll" -Recurse -ErrorAction SilentlyContinue | Select-Object -ExpandProperty FullName

DLL이 발견되면 해당 폴더(예: C:\tools\fluidsynth\bin)를 시스템 환경 변수 PATH에 추가한 뒤 셀라를 재시작하세요.

PATH 추가 방법 (관리자 PowerShell):

[System.Environment]::SetEnvironmentVariable(
    "PATH",
    [System.Environment]::GetEnvironmentVariable("PATH","Machine") + ";C:\tools\fluidsynth\bin",
    "Machine"
)

9. SoundFont(.sf2) 설치 필수 — MIDI 합성

SoundFont는 FluidSynth가 MIDI 음표를 실제 악기 음색으로 렌더링하는 악기 샘플 뱅크입니다. 셀라에는 포함되어 있지 않으므로 별도로 다운로드합니다.
A SoundFont is an instrument sample bank FluidSynth uses to render MIDI. Not bundled — download separately.

추천 무료 SoundFont — GeneralUser GS (~29 MB)

  1. schristiancollins.com/generaluser.php 에서 다운로드합니다.
  2. 압축을 해제하고 GeneralUser GS.sf2 파일을 꺼냅니다.
  3. 아래 배치 위치 중 하나에 복사합니다.

SoundFont 파일 배치 위치 (우선순위 순)

우선순위 경로
1 (최우선)%AppData%\Selah\soundfonts\
2<앱 폴더>\soundfonts\
3C:\Program Files\FluidSynth\
4C:\soundfonts\

권장 위치: %AppData%\Selah\soundfonts\
탐색기 주소 표시줄에 위 경로를 그대로 붙여넣으면 폴더가 열립니다. (셀라가 첫 실행 시 자동으로 생성합니다.)

기타 추천 무료 SoundFont

  • FluidR3_GM.sf2 (~141 MB) — 다양한 Linux 배포판에 포함, 광범위하게 사용됨
  • MuseScore_General.sf2 — MuseScore에 번들로 제공, 고품질·대용량
