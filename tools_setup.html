1. Python 설치 필수 — MR 추출

스템 분리(MR 추출) 기능에 Python 3.10 이상이 필요합니다.
Python 3.10 or later is required for stem separation (MR extraction).

⚠ Windows 스토어(Microsoft Store) Python은 사용하지 마세요.
스토어 버전은 백그라운드 실행 시 제대로 동작하지 않습니다. 반드시 python.org 에서 설치하세요.

설치 방법

  1. python.org/downloads 에서 Python 3.10 이상(최신 안정 버전 권장)을 다운로드합니다.
  2. 설치 프로그램 첫 화면에서 "Add Python to PATH" (또는 "Add python.exe to PATH") 옵션을 반드시 체크합니다.
  3. 설치를 완료한 뒤 셀라를 재시작합니다.

설치 확인 — 명령 프롬프트(CMD) 또는 PowerShell에서:

python --version

Python 3.x.x가 출력되면 정상입니다. 오류가 나면 PATH 설정을 확인하세요.

2. ONNX Runtime 설치 필수 — MR 추출

AI 모델 추론(inference)에 ONNX Runtime 및 관련 패키지가 필요합니다.
ONNX Runtime and related packages are required to run the AI separation model.

설치 방법

명령 프롬프트(CMD) 또는 PowerShell을 열고 다음 명령을 실행합니다:

pip install onnxruntime numpy scipy

pip 명령을 찾지 못하는 경우 아래 명령을 사용하세요:

python -m pip install onnxruntime numpy scipy
⚠ 가상 환경(venv, conda 등)을 사용 중이라면 셀라가 해당 환경을 자동으로 인식하지 못할 수 있습니다. 가상 환경 없이 전역(global) Python 환경에 설치하거나, 셀라 실행 전에 가상 환경을 활성화하세요.

설치 확인:

python -c "import onnxruntime; print(onnxruntime.__version__)"

3. noisereduce 설치 필수 — 노이즈 감소

편집 > 노이즈 감소 기능에 noisereduce 라이브러리가 필요합니다. 모델 파일 없이 pip 한 줄로 설치됩니다.
The "Reduce Noise" feature requires the noisereduce library. No model file — install with one pip command.

설치 방법

명령 프롬프트(CMD) 또는 PowerShell을 열고 실행합니다:

pip install noisereduce

pip을 찾지 못하면:

python -m pip install noisereduce

설치 확인:

python -c "import noisereduce; print('noisereduce OK')"
⚠ 가상 환경(venv) 사용 시 주의
pip 출력에 .\\.venv\\ 또는 site-packages 경로가 보인다면 가상 환경에 설치된 것입니다. 셀라는 가상 환경을 자동 감지하지 않으므로 전역(global) Python에 재설치해야 합니다.

전역 Python에 설치하는 방법: 
py -m pip install noisereduce
(py 명령은 Windows Python Launcher로 전역 Python을 직접 호출합니다.)

4. FFmpeg 설치 필수 — MP3·MP4 임포트 & MR 추출

MP3, MP4, MKV 등 비(非)WAV 파일 임포트와 MR 추출(오디오 디코딩/인코딩)에 FFmpeg가 필요합니다.
FFmpeg is required for importing non-WAV files and for audio decode/encode during MR extraction.

방법 A — winget 사용 (가장 간단, Windows 10/11)

명령 프롬프트를 관리자 권한으로 열고 다음 명령을 실행합니다:

winget install Gyan.FFmpeg

설치 후 명령 프롬프트를 새로 열고 셀라를 재시작하세요.

방법 B — 수동 설치

  1. ffmpeg.org/download.html → Windows builds 링크에서 최신 빌드를 다운로드합니다.
  2. 압축을 해제하고, 내부의 bin\ 폴더 경로(예: C:\ffmpeg\bin)를 시스템 환경 변수 PATH에 추가합니다.
    • 시작 → "환경 변수" 검색 → "시스템 환경 변수 편집" → Path → 편집 → 새로 만들기
  3. 명령 프롬프트를 새로 열고 확인합니다:
ffmpeg -version

5. 분리 모델 다운로드 필수 — MR 추출

위의 1~3 단계 완료 후, 셀라 내 모델 관리자에서 AI 모델 파일을 다운로드합니다. (~303 MB)
After completing steps 1–3, download the AI model from Selah's built-in Model Manager.

다운로드 방법

  1. 셀라 메뉴 → 도구 > 모델 관리자를 엽니다.
  2. 목록에서 Hybrid Transformer Demucs를 선택합니다.
  3. "모델 설치 / 다운로드" 버튼을 클릭합니다.
  4. 설치 로그에 "모델 설정 완료" 메시지가 나타나면 "새로고침" 버튼을 클릭합니다.
📥 모델 파일 크기: 약 303 MB. 다운로드 중 인터넷 연결이 유지되어야 합니다.
