使用指南

什么是 Selah？

Selah 是一款用于敬拜音乐准备、项目式音频编辑和 AI 辅助音轨分离的 Windows 桌面应用程序。专为小型教会、宣教工场和敬拜团队设计，帮助他们从录音或敬拜媒体中实用地制作以伴奏为主的音频资料。Selah 是完全离线运行的免费开源软件（GPLv3）。

开始使用

首次启动 Selah 时，会显示欢迎界面。点击 ＋ 新建项目 创建新项目，或点击 📂 打开项目 加载已有项目。创建新项目时，选择名称和采样率——视频/流媒体工作推荐 48 kHz，音乐文件推荐 44.1 kHz。

项目管理

项目将所有轨道、片段和设置保存在同一个文件夹中。

Ctrl+S 保存  |  Ctrl+Shift+S 另存为  |  Ctrl+O 打开

关闭时若有未保存的更改，系统会提示是否保存。

导入音频

使用 文件 → 导入音频/视频 可导入 WAV、MP3、FLAC、MP4 等多种格式。WAV 以外的格式需要安装 FFmpeg。同时选择多个文件时，每个文件会分别添加为独立轨道。

时间轴编辑

移动片段 — 拖动片段更改其位置。
分割片段 — 将播放头定位到目标位置，按 S 键。
删除片段 — 选中片段后按 Delete 键。
缩放 — 使用 Ctrl+滚轮 放大/缩小。
吸附 — 按 N 键切换吸附。

在轨道标题栏中使用音量滑块、M（静音）和 S（独奏）按钮控制各轨道。

AI 音轨分离 / MR 提取

使用 AI 音轨分离可以从音乐中去除人声，或分离鼓、贝斯和乐器。

1. 打开 工具 → 模型管理器 并安装分离模型。（需要 Python 3.10+ 和网络连接）
2. 在时间轴中选择要处理的片段。
3. 执行 编辑 → 分离音轨。
4. 分离完成后，每个音轨将自动添加为新轨道。

降噪： 编辑 → 降噪。（需要：pip install noisereduce）

导入乐谱

从扫描或拍摄的乐谱图像（PNG、JPG、TIFF）自动生成各乐器的音频轨道。

1. 选择 文件 → 导入乐谱图像。
2. 选择乐谱图像文件，点击 开始识别。
3. 识别完成后，从乐器列表中选择所需乐器（★ = 自动推荐）。
4. 点击 插入轨道，每个乐器将自动添加为新轨道。

此功能需要 Python 包（oemer、music21 等）、FluidSynth 和 SoundFont (.sf2)。请参阅下方 必要工具安装 ⑦⑧⑨。

导出

编辑完成后，使用 文件 → 导出为 WAV（Ctrl+E）将整个时间轴渲染为 24-bit WAV 文件。静音轨道不会包含在输出中。如果节拍器处于开启状态，它将被包含在输出中——导出前请确认。

必要工具安装

音轨分离（MR 提取）和降噪功能需要安装以下工具。基本 WAV 编辑和导出无需额外安装。

① 安装 Python — 必须（所有 MR 提取功能）

从 python.org/downloads 下载 Python 3.10 或更高版本。安装时必须勾选 "Add Python to PATH"。安装完成后重启 Selah。

⚠ 请勿使用 Microsoft Store 版本的 Python——它无法正常作为后台进程运行。请务必从 python.org 安装。
验证安装（命令提示符）：
python --version

② 安装 ONNX Runtime — 必须（Demucs 模型）

pip install onnxruntime numpy scipy
如果找不到 pip 命令：
python -m pip install onnxruntime numpy scipy
⚠ 使用虚拟环境（venv、conda）时：Selah 不会自动检测虚拟环境。请安装到全局 Python，或使用 py -m pip install。

③ 安装 audio-separator — 推荐（人声分离）

能识别比 Demucs 更广泛的人声范围，支持各种音域和演唱风格。如需 GPU 加速，将 [cpu] 替换为 [gpu]。（需要 CUDA 12.x）

pip install "audio-separator[cpu]"

④ 安装 noisereduce — 必须（降噪）

pip install noisereduce

⑤ 安装 FFmpeg — 必须（MP3/MP4 导入）

方法 A — 以管理员身份打开命令提示符（推荐）：

winget install Gyan.FFmpeg

方法 B（手动）— 从 ffmpeg.org/download.html 下载 Windows 版本，将 bin\ 文件夹路径添加到系统 PATH 环境变量。

⑥ 下载 AI 分离模型

安装上述工具后，打开 工具 → 模型管理器 下载 AI 模型。

Hybrid Transformer Demucs (htdemucs)
4音轨分离 · 303 MB · 需要 ONNX Runtime
UVR MDX-NET Vocal FT
人声/伴奏 2音轨 · 65 MB · 需要 audio-separator

选择模型后点击 安装模型 / 下载，完成后点击 刷新。

⑦ 安装 OMR 包 — 必须（乐谱导入）

pip install oemer music21 mido Pillow scipy
oemer 首次运行时会自动下载模型权重（需要网络连接）。

⑧ 安装 FluidSynth — 必须（乐谱 → 音频合成）

以下两个步骤都必须完成。

步骤 1 — 安装原生库（libfluidsynth-3.dll）：

choco install fluidsynth

或从 fluidsynth.org 下载 Windows 安装程序并运行。Selah 会自动检测 DLL，无需配置 PATH。

步骤 2 — 安装 Python 包装器：

pip install fluidsynth
⚠ 仅安装 pip 包而跳过原生安装程序将导致合成时出错。两个步骤都必须完成。

⑨ 安装 SoundFont (.sf2) — 必须（乐器音色）

SoundFont 是 FluidSynth 用于将 MIDI 渲染为真实乐器音色的乐器采样库。

推荐免费 SoundFont： GeneralUser GS（约 29 MB）
从 schristiancollins.com/generaluser.php 下载，将
GeneralUser GS.sf2 复制到以下路径：

%AppData%\Selah\soundfonts\
在 Windows 资源管理器地址栏中直接输入或粘贴上述路径即可打开文件夹。

伦理使用准则

Selah 是为支持敬拜事工而创建的。用户有责任自行确认所处理音频的版权状态。

项目目的

Selah 是一款以事工为导向的音频软件，旨在支持基督教敬拜准备、小型教会音频工作流程以及宣教伴奏制作。优先考虑非商业、以服侍为中心、本地化的使用场景。

尊重权利与所有权

用户必须尊重作曲家、作词家、演奏者、制作录音的教会和事工机构、出版商及版权持有者的权利。技术上可行并不代表在法律或道德上是被允许的。

负责任使用示例

✓ 使用合法获取的敬拜音频为小型敬拜团队制作伴奏
✓ 整理事工录音用于教会内部排练
✓ 对您有权处理的音频进行音轨分离测试

禁止使用示例

✗ 未经许可重新分发受版权保护的敬拜录音
✗ 将分离的音轨进行商业包装再销售
✗ 在没有合法权利的情况下去除人声或提取音轨
✗ 在没有授权的情况下误导他人认为已获得某教会或事工机构的认可
